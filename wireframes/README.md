## Design Guidelines

### Color Palette
- **Primary:** #FF69B4 (Pink) - Feminine, caring, PCOS awareness color
- **Secondary:** #4A90E2 (Blue) - Trust, healthcare
- **Accent:** #50C878 (Green) - Wellness, growth
- **Background:** #F5F5F5 (Light Gray)
- **Text:** #333333 (Dark Gray)

### Typography
- **Headings:** Poppins, Bold
- **Body:** Open Sans, Regular
- **Buttons:** Poppins, Medium

### Design Principles
1. **Empathetic Design** - Warm, supportive, non-clinical
2. **Accessibility** - WCAG 2.1 AA compliant
3. **Mobile-First** - Responsive on all devices
4. **Minimal Cognitive Load** - Simple, intuitive navigation

## Responsive Design Breakpoints

- **Mobile:** 320px - 767px

- **Desktop:** 1024px+

## Interactive Elements

### Buttons
- **Primary:** Filled pink, white text
- **Secondary:** Outlined pink, pink text
- **Hover:** Darker shade, slight scale
- **Disabled:** Gray, reduced opacity

### Forms
- **Input Fields:** Rounded borders, subtle shadow on focus
- **Validation:** Real-time with error messages
- **Success:** Green checkmark icon

### Cards
- **Shadow:** Soft drop shadow
- **Hover:** Lift effect (translateY)
- **Clickable:** Pointer cursor
